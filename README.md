This is a full-stack Airbnb Clone web application developed using Node.js, Express, MongoDB, and EJS.
The project allows users to register, login, create property listings, view details, and post reviews.
It follows the MVC architecture and focuses on real-world web development practices.


 Tech Stack

Frontend: EJS, HTML, CSS, JavaScript, Bootstrap,

Backend: Node.js, Express.js,

Database: MongoDB (MongoDB Atlas)
